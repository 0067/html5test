About these scores

Each of the browsers above have been used to run the HTML5 test. The browsers are run with their default settings. If you don't see your browser in these tables don't worry, these tables will be updated regularly. It is not possible to list all versions of all browsers, but there are more detailed results available at browserscope.org.
